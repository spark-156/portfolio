# Portfolio
Portfolio website made in React with a NodeJS backend and a MongoDB database and hosted in docker behind an nxgin reverse proxy.

# Usage
Dependecies:

1. Docker
2. Docker-compose

Steps to run website:
1. Git clone this repo
2. CD in to cloned repo folder
3. run ```make buildProd``` in the command line
4. run ```make startProd``` in the command line
5. go to localhost:80 to see website
